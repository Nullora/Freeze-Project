# Security Policy

FreezeOS is a hobby operating system developed by Clashnewbme and volunteers. It is unreleased software still in early development, and so bugs and vulnerabilities can be safely disclosed publicly. If you find an issue, we would prefer if you could report it as a [GitHub issue](https://github.com/Clashnewbme/Freeze-Project/issues/new). If your issue is small please check our [Documentation](https://freezeos.org/documentation.html) to find a possible solution.
