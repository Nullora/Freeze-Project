# Freeze Project

> A simple, easy to set up operating system

"FreezeOS is a clean, designed to be Lightweight, simple, and customizable **OS."

---

## Features

- Many documents created by users
- Small size, big performance
- Easy to use
- Open-source
- Guntenberg Library
